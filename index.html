Oracle de Traversée
Le texte apparaîtra ici après génération.
Outil séparé, isolé de ton site client : les 34 cartes, leurs images et leurs mots d'orientation sont repris de ton index.html actuel, mais les images sont chargées directement depuis ton repo oracle-traversee sur GitHub, sans copie. Ce fichier est donc autonome : tu n'as qu'à déposer ce seul fichier dans un nouveau repo à toi, sans rien toucher au repo de tes clientes.

Climat de la semaine : l'outil calcule en silence la phase lunaire, les rétrogradations et les tensions/harmonies planétaires du jour, entièrement en local, sans API externe. Ce climat ne sort jamais dans le texte, il ne fait qu'orienter légèrement le vocabulaire de l'IA quand plusieurs formulations sont possibles. Les cartes restent toujours la seule vérité du tirage.

L'image (juste les 3 cartes, format carré Instagram) et le texte (légende) sont maintenant séparés : télécharge l'image d'un côté, copie le texte de l'autre. Si jamais une carte affiche "Image manquante" à l'usage, ou si la génération de texte échoue, tu peux toujours écrire le texte toi-même directement dans la zone (elle reste éditable).
